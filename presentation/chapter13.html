public void testAliasing() {
    List numbers = Arrays.asList(1,2,3,4,5);
    List anotherNumbers = numbers;

    anotherNumbers.add(10);

    assertThat(anotherNumbers.size(), is(6));
    assertThat(numbers.size(), is(5));
}

별칭(aliasing) 문제

다른 참조를 통해 객체에 접근하는 쪽에서 내부 데이터가 변경될 가능성이 언제나 존재한다.
문제가 생길 경우 프로그램 전체에서 데이터 갱신 사실을 추적하는게 어려워진다.
